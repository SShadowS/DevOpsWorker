# Translation Files (.xlf)

**Never modify `.xlf` translation files.** These are managed by a separate translation pipeline and must not be touched by this agent.

Your responsibility is to ensure English captions, tooltips, and labels are correct in the AL source code (`.al` files). The translation system propagates those English strings into the `.xlf` files automatically.

If the plan mentions translation files, ignore that part — focus only on AL source changes.
